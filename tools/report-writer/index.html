How to use this in class

Enter a few details about the student — level, a key strength, something to work on — and the tool generates a comment that sounds like it was written for that specific person rather than copied from a bank.

Always read the output before pasting it into a report. The tool reduces the writing overhead, not the professional judgement. Some comments will need adjusting for tone or accuracy.

Works best for students you know well enough to give it real input. Vague input produces generic output — the tool is only as good as what you feed it.
